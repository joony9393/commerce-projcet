# SpringBoot - commerce project
회원가입을 통해 회원을 받고 장바구니 기능을 구현, 상품 재고를 관리하는 프로젝트

## 사용기술
- SpringBoot
- SpringDataJPA
- SpringSecurity
- Redis
- MySQL

## 프로젝트 기능
### 회원 관리
- [ ] 회원 가입, 로그인, 회원정보 수정
- [ ] 로그인
- [ ] 탈퇴

### 장바구니
- [ ] 장바구니 등록
- [ ] 수량 수정
- [ ] 제거
  

### 재고관리
- [ ] 특정 상품 재고 확인
- [ ] 새로운 상품 추가
- [ ] 특정 상품 재고 업데이트
- [ ] 특정 상품 재고 삭제
